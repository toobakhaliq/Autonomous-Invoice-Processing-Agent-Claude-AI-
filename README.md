# AI Agent Project

## Overview
This project contains an AI-powered agent built to extracts invoice data, categorizes expenses, validates compliance with purchasing policies, and produces structured outputs for financial workflows. Designed for automation of finance operations and procurement compliance.

## Features
- Autonomous task execution
- LLM integration
- API integrations
- Logging & memory
